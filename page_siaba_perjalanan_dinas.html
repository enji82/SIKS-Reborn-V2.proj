Data Perjalanan Dinas
Informasi
  • Halaman ini digunakan untuk pengajuan Perjalanan Dinas.
  • Batas waktu pengajuan adalah 1 hari sebelum tanggal pelaksanaan perjalanan dinas.
  • Format Surat Tugas dapat diunduh di menu Tata Naskah Dinas.
  • Apabila Surat Tugas Final belum ada dapat menggunakan Surat Tugas sementara, dan wajib diperbarui apabila Surat Tugas Final sudah ada.
  • Klik tombol Tambah SPT untuk memulai pengajuan baru.
  • Klik tombol 💬 Baca Catatan untuk melihat instruksi/alasan dari admin jika status Disetujui, Revisi, atau Ditolak.
  • Data divalidasi menggunakan nomor surat, sehingga tidak diperbolehkan mengisikan Surat Tugas dengan nomor surat yang sama. Untuk menambahkan Nama ASN pada nomor surat yang sudah terdata, gunakan tombol Edit.
  • Halaman ini digunakan sebagai alat bantu admin Korwil untuk menginput ke aplikasi Siaba, jika status Disetujui maka sudah berhasil diinput.
